# Python Flask Web App & Chat Bot

## Setup & Installation

Make sure you have the latest version of Python installed.

```bash
git clone https://github.com/KPasumarthy/KPPythonChatBot
```

```bash
pip install -r requirements.txt
```

## Running The App

```bash
python main.py
python bot.py
```

## Viewing The App

Go to `http://127.0.0.1:5000`
